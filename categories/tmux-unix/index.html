Sun, Nov 27, 2016 tmux is an awesome tool! I’ve used it literally constantly for about a year now. And guess what! I didn’t know about some very basic commands. List Sessions Command: <prefix>s This shows you an interactive screen where you can which session you want to be in: (0) + 0: 3 windows (attached) (1) + 1: 1 windows Switch Session Command: <prefix>( # or <prefix>) Move back and forth between sessions.
