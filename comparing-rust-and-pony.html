Comparing Rust and Pony

Rust and Pony are two relatively new languages that both use type systems to prevent data races with a focus on performance. How do they compare?

Bias

I'd like to first say I think both of these languages are super cool. This is not at all a contest; I just like exploring languages!

Similarities

Why are these languages worth comparing?

Hello World

Rust:

fn main() {
  println!("Hello world");
}

Pony:

actor Main
  new create(env: Env) =>
    env.out.print("Hello, world")

So this actually exposes a lot of the differences between the languages. First of all, there's the syntax. Rust looks like a classic C program, Pony looks like a Python program.

Right of the bat we get a key observation: Rust is targeting system programmers and Pony is aimed at somewhat higher level programming.
